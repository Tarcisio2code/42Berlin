# 42 Berlin - C Piscine - Schell00 - Exercise 00


**Subject**
>_Create a file called z that returns "Z", followed by a new line, whenever the command
cat is used on it._
>
>**_Example_**
>
>_&>cat z_   
>_Z_   
>_&>_   
>

<br>

**Commands used**   
>_Echo or Cat._

<br>

**Solution**
>```diff 
>Echo Z > z
>```   
>_or_
>```diff 
>Cat > z   
>type Z and press enter and then ctrl+d to finish.
>``` 

<br>

**Result**
>```diff 
>cat z
>```
>_Z_
