# 42 Berlin - C Piscine - Schell00 - Exercise 00


**Subject**
>_Create a file called z that returns "Z", followed by a new line, whenever the command
cat is used on it._
>
>**_Example_**
>
>_&>cat z_   
>_Z_   
>_&>_   
>

<br>

**Commands used**   
>_Echo or Cat._

**Solution**   
>_**Echo** Z > z_   
>
>_**Cat** > z_   
>_type Z and press enter and then ctrl+d to finish._
